## 与篮球

说到篮球，其实接触了也挺长一阵子了，玩的比较多的时间集中在初高中，上大学之后也不咋玩了，完全就是天天在宿舍打游戏，偶尔身体有点乏了，才出去跑跑步。出来工作之后，会一个月半个月跟着部门人去篮球场打打球，不得不说，球场的费用也是真的贵，而且感觉场地也不咋地，全是灰。 本人的篮球水平也只是打打野球的水平，比赛打得不多，也没啥亮眼表现，所以这篇文章大佬们看个乐呵就行，当然我也希望能启发到你。 接触篮球的时候有几位好朋友带着玩，能勉强学到些技术，正巧当时网络也发达起来，网上开始有很多篮球教学，我也开始了我的痛苦学习摸索之路。现在想想真的浪费了很多时间在一些无用的练习上，导致了球技非常畸形，成了一个运球花里胡哨，终结一坨狗屎的坑逼，回顾之前学的乱七八糟的各种技术，也完全没有任何头绪，比如思考这项技术为什么会诞生，什么时候用，种种，也就导致了每次打球就像是一个没有头脑的出招机器，非常拉跨，现在回想起来都有点想给我一拳。所以各位想入坑篮球的，建议还是加个球队，让教练带着练，一个技术一个技术慢慢打磨，宁可少学，学好，也不要乱学。多练终结，少练运球和一些花里胡哨的东西。

:::primary no-icon
投篮准，突破快，上篮稳的人最难防守，注重基本功的练习非常重要
:::

## 一些学到的东西分享

下面我对我曾经接触到的技术，做些简单罗列（摸鱼无聊写的）。篮球的技术其实非常之多，但是篮球是一项吃反应的运动，要根据场上瞬息万变变化进行不断调整的运动，所以下列技术能吃遍大部分情况，但更多的情况，也可以根据自己的理解进行补充和调整，找到最适合自己的球风。

### 终结

终结是篮球运动中最重要的部分，因为打篮球纯粹就是把球放进篮筐，所以对终结的训练，是远重要于其他部分的

#### 终结方式

终结的方式以离篮筐远近按半圆划分可分为

1. 上篮区（合理冲撞区）
2. 抛投区（低位）
3. 中投区
4. 三分区

四种终结方式的难易度依次递增，所以在进攻的选择上，往往是由上到下，取决于防守实际情况来进行终结。
比如过完人篮下无人则可以轻松选择上篮；过完人篮下如果站了人无法进行上篮则可以选择抛投；如果上篮区和抛投区都被封锁，则可以选择中投或者三分。
四种终结在空位时都要保证分别有10成、9成、7成、5成命中率才算合格

:::success no-icon
投篮也可以
野球场大部分防守者都会放投，这种情况下强行突破出手完成上篮，往往不如一个空位中投或者三分命中率高，所以练好投篮很重要。
:::

:::success no-icon
传球也可以
终结选择可以实际情况应灵活变通，实在难以终结也可以将球传出去，篮球不是一个人的运动，要多转移球多跑动才能更好地完成终结.
:::

#### 上篮选择

1. 直接上篮--没什么好说的
2. 假投上篮--在即将杀到篮下，防守者追得很紧的时候，停球进行假上篮/抛投，在晃飞防守者后再进行终结，无论是半场还是全场快攻，都是一个屡试不爽的必备技能包
3. 拉杆上篮（空中假投上篮）--在杀到篮下进行直接上篮但是受阻时，空中假上篮晃飞防守者躲避封盖，再改变出手位置完成上篮。比较费腰，但也很实用，
4. 对抗上篮--在上篮过程与防守者进行对抗，顶开空间后进行上篮，n+1必学动作，个人没有训练对象，基本没用过

:::success no-icon
传球也可以
很多时候强行出手并不是一个好的选择，突破造成杀伤，吸引防守后把球传给空位的队友也许更为正确
:::

#### 上篮脚步

1. 一步
2. 跳步
3. 转身
4. 欧洲步
5. ...

上篮的变种脚步繁多复杂，实战中阅读防守并运用的场景也比较苛刻，可以选择性进行学习练习，不是重点练习的对象。

### 阅读防守（创造终结机会）

:::primary no-icon
阅读防守遵循防投则突、防突则投原则
所以进攻时基本上可以选择左路、右路突破，或是投篮、后撤投篮
:::

### 创造空间

很多防守情况下，如果进攻能在持球阶段就完成的话，不需要多此一举去运球完成终结，尽量减少无效运球次数，确保每次运球都是有目的性的
如果防守人员很好地防守住你的位置，那么就需要离开这个防守位置，去创造投篮或者突破的空间

Pull Dribble（悬浮运球）就是一个很好地拉开空间的方式，或者其他移动方式，你第一步需要离开被防守住的位置，调动你的防守人员，根据防守人员进行下一步进攻策略

### 节奏变换

节奏变换是甩开防守者最重要的一个技术，如果只是一昧地进行无意义运球动作以及毫无节奏变化的启动突破，那么在双方身体素质差不多的情况下很难拉开终结空间，强行完成终结往往会错失该次进攻得分的机会，篮球节奏瞬息万变，每次进攻都至关重要

节奏变换分为由静到动的**犹豫（Hesitation）** 和由动到静的 **急停**

### 核心技能

#### 犹豫（Hesitation）

犹豫指的是在突破启动前犹豫一下，也可以是做一个假投动作，使防守人员的节奏与你错开的同时可能还因忌惮你的投篮而吃晃

#### 急停

急停指的是在高速运球突破时突然刹车的一个动作，使得防守人员也被迫急停，急停后带来的节奏差甚至失位会使得接下来的进攻选择更加丰富且极具杀伤力。

#### 变向

变相是遭到防守人员提前卡住突破位置时，改变突破方向的一种技术，启动速度或者突进速度越快，防守人员更容易因为提前设防吃晃

:::success no-icon
变相的方式可以为体前、胯下、背后、转身等等
体前变向常用于与对手距离一个手臂身位快速转移球到另一侧
胯下变向用于与对手距离半壁身位，将脚跨过防守者并突破
背后运球和转身用于对方贴防、断球时，快速转移球至安全侧
:::

#### 后撤步

后撤步可以看作是急停跳投拉开空间版

### 个人进攻框架

下面讲讲个人单打进攻思路
进攻的选择为左路、右路突破和投篮，持球和运球进攻都是一样的基于三者选择，阅读防守也是基于三种选择来执行创造空间
创造空间的核心节奏技能是犹豫、变向、急停，辅以后撤来进行

#### 持球进攻

进攻最好在持球阶段就解决
持球面对防守人时可以用三威胁阅读防守，如果防守防突留出空间则果断选择投篮出手，如果贴上来防投，则可以尝试将脚越过防守者尝试左路或者右路进行突破，手感不错时也可以原地撤步投篮

#### 运球进攻

Pull Dribble横向移动调动防守后阅读防守
0. 防守者贴防上来试图断球--做一个背后或者转身到另一侧突破
1. 防守没有跟上站住常规防守位--做个犹豫后突破
2. 防守选择封堵你的突破路线，此时具备投篮空间--直接投篮完成终结
3. 防守者跟防上来，处在一个常规防守点位
  1）做一个假突变向突破
  2）做一个犹豫假投突破

突破之后如果没有摆脱防守，做一个急停
0. 防守者在你突破时主动上来找对抗贴防、试图断球--做一个背后或者转身到另一侧突破
1. 急停后防守者失位但没有很及时扑上来--直接急停跳投
2. 急停后防守者失位及时扑上来--属于是跳投做一半，可以转为突破，选择有突破空间的一侧下球
3. 急停后想要直接投篮完成，不跟他罗里吧嗦--直接急停后撤步跳投
4. 对手在领略了你的后撤步跳投后，对此招有所防备--急停后撤步做一个假投再突破

:::danger no-icon
如果你在上述一系列操作后还是未能摆脱防守，那么你需要通过挡拆换一个好打的防守者，或者将你的球交给其他队友来发起进攻，或者尝试一些战术来解决问题，也许会轻松得多
:::

:::warning no-icon
上面介绍的创造终结机会仅限于单打强吃的情况，并不是得分的最优解，实际篮球比赛中直接**终结**的机会非常多，使用无球跑动和挡拆来进行得分才是最优解。在进攻机会不好时，把球传出去或者突破分球给其他队友也是更好的选择，让球轮转起来并进行更多的跑动，才能使得分更加轻松，参考斯蒂芬库里的打球方式
:::

## 训练计划

训练的方式是多种多样的，特别是对身体素质的训练是打好篮球的基础，包括体能训练和力量训练等

下面简单写一些针对本篇提到的技术要点来制定的训练内容

### 终结训练

#### 上篮训练

上篮相对投篮虽然更简单稳定，但是上篮相关训练也很重要，很多人并不能做到每次上篮都稳稳打进，受到实战中防守者的对抗干扰、上篮角度刁钻、突破速度快上篮力度失控等等因素影响，所以扎实的上篮训练是必不可少的，以下训练一组为进一个球

1. 高低手上篮-左边左手高低手各5组（右边同理）
2. 内外侧手上篮-左边左手内外侧手各5组（右边同理）
3. 反手上篮-左路突破左右手上反蓝各5组（右边同理）
4. 抛投-左侧突破空心抛投、打板抛投各5组（右边同理）
5. 技巧上篮-左右侧突破假投上篮各5组；左右侧突破拉杆上篮各5组

#### 投篮训练

投篮训练内容基于库里大师课

1. 篮球场5个方向4个距离共20个点位，各5组，共100组
2. 5个方向3个距离点位（三分、中投、低位）左右手运一下投篮各3组，共90组
3. 同上运球迈一步急停跳投，左右各3组，共90组
4. 3个方向3个距离点位左右突破做后撤步跳投（三分、中投、低位）各5组，共90组

## 最后

上述篮球技巧仅参考个人球风所制定部分技巧，也并没有展开进行详细介绍，如上篮的一些手法、投篮的发力等等，只是提供一些简洁高效的技巧和场上的进攻时阅读防守见机行事的思路
篮球的技术远不止这些，还有诸如内线脚步、后仰跳投、背身等技术动作，感兴趣也可以自行学习了解。
写这篇文章的目的主要是对个人在场上时面对各种场景

:::primary no-icon
篮球得分其实并没有那么难，难的是防守而非进攻，如开头所述，如果你的投球准，速度快，上篮稳，那么你将是一个抢手的得分机器，所以练球的时候，要把时间更多的用在终结上，把投篮练准，把上篮练稳，然后再去思考无球跑位以及一些简单的挡拆配合，最后再去练一些单打常用的控球晃人技能，当然基础的运球是必不可少的，否则就相当于自废双足了，当运球不能满足的时候，再去进行控球节奏练习。
:::
